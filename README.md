### 🚀 "1ms의 가치를 아는, 타협하지 않는 엔지니어"

> 안녕하세요, 개발자 이재호입니다. 인디게임 스타트업을 운영하며 제품 기획부터 개발, 운영까지 전 과정을 경험했습니다. 단순히 코드를 작성하는 것을 넘어 비즈니스 목표를 기술로 실현하는 데 익숙합니다. 20회 이상의 프로덕션 릴리즈를 직접 관리하며 안정적인 배포 체계를 구축해 왔으며, 기술적 난제 앞에서는 끝까지 답을 찾아내려 노력합니다. 동료들과 원활하게 소통하며, 내 이름이 걸린 프로젝트는 끝까지 책임진다는 오너십을 가지고 일합니다.

- **Core Value**: 화려한 로직보다 '멈추지 않는 시스템'의 가치를 믿습니다. 1,000번의 알고리즘 해결 과정에서 얻은 검증 습관으로 무결성 소프트웨어를 지향합니다.
- **Expertise**: C/C++ 기반의 저수준 최적화와 Unreal Engine을 활용한 고성능 시뮬레이션 환경 구축에 강점이 있습니다.
- **Philosophy**: 개발자는 책상이 아닌 현장에서 답을 찾아야 한다고 믿습니다. 3년간 인디 게임 팀을 이끌며 기술을 넘어선 서비스의 완결성을 증명해 왔습니다.
